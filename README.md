# katialafono.cl

Repositorio de la web de producción de **Katia Domínguez**, fonoaudióloga en Chillán.

**Sitio web:** <https://www.katialafono.cl>

## Stack

- Next.js
- App Router
- TypeScript
- Vercel

## Desarrollo local

```bash
npm run dev
```

Abrir `http://localhost:3000`.
